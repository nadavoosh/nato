# nato
`nato` is a simple tool to get [NATO spelling alphabet](https://en.wikipedia.org/wiki/NATO_phonetic_alphabet) translations.

## installing
`go install`

## usaage
```
$ nato hi there 123
HOTEL INDIA
TANGO HOTEL ECHO ROMEO ECHO
ONE TWO THREE
```

The bare `nato` commamnd will print the alphabet:
```
$ nato
a: ALFA
b: BRAVO
c: CHARLIE
d: DELTA
// etc...
```
